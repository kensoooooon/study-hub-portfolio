# Django Study Hub — ポートフォリオ資料

## エグゼクティブサマリー

Django Study Hub は、LINE Messaging API と OpenAI（ChatGPT）を組み合わせた **AI駆動の教育SaaSプラットフォーム** です。英単語・リスニング・リーディング・数学など複数教科の学習機能を提供し、スペースドリピティション（SuperMemo-2）アルゴリズムによる科学的な記憶定着を実現しています。Google Cloud Platform（GCP）上で動作するマルチテナント型アーキテクチャを採用し、組織・教室・学生という3層の権限管理を実装しています。

## 3. アーキテクチャ設計のポイント

### 3-1. マルチテナント型SaaS設計

4層のロール階層（OrganizationAdministrator → ClassroomAdministrator → Teacher → Student）により、複数組織が同一インスタンス上で独立したデータ空間を持てる設計を実現しています。

```
Organization（塾・学校など）
  └── Classroom（クラス）
        ├── Teacher（教師）
        └── Student（生徒）
```

各モデルのQuerySetに独自メソッドを実装し、ロールに基づいたデータフィルタリングをアプリ全体で一貫して適用しています。

### 3-2. SuperMemo-2アルゴリズムの実装

単純な学習履歴の記録にとどまらず、記憶科学に基づくスペースドリピティション（SuperMemo-2）を全トレーナーモジュールに共通実装しました。

- `ease_factor`（記憶の容易さ係数）
- `interval`（次回復習までの間隔）
- `next_due_at`（次回復習予定日時）

各ユニットの学習品質スコア（0〜5）から自動でスケジュールを計算し、忘却曲線に沿った最適なタイミングで復習を促します。

### 3-3. エンベロープ暗号化によるLINEチャネル管理

組織ごとに異なるLINEチャネルの認証情報（CHANNEL_SECRET, ACCESS_TOKEN）を、Cloud KMSを用いたエンベロープ暗号化で保管しています。

```
平文シークレット
    │ DEK（データ暗号化鍵）で暗号化
    ▼
暗号文（LineChannelKeyBundle）
    │ KEK（Cloud KMS管理鍵）でDEKを暗号化
    ▼
Cloud KMSに安全に保管
```

鍵のローテーションにも対応し（`is_active`フラグ）、セキュリティ基準を満たした設計を実現しています。

### 3-4. 非同期リマインダー配信（Pub/Sub）

スタディリマインダーはCloud Pub/Subを活用した非同期アーキテクチャで実装しています。

```
trigger_reminders.py → Pub/Sub Topic → listener.py → LINE API
     （スケジューラ）       （キュー）    （購読プロセス）
```

Webプロセスとは独立した`listener.py`がメッセージを受信・処理するため、リマインダー配信がWebアプリのレイテンシに影響しない設計です。

### 3-5. AIによるコンテンツ自動生成

`processors/` モジュールにOpenAI API連携を集約しています。

- **reading_passage_generator.py**（531行）: リーディング問題文・設問・選択肢の自動生成
- **listening_passage_generator.py**（590行）: リスニング問題文と音声コンテンツの生成
- **chat_processor.py**: 学習者とのChatGPTによる会話応答
- **example_sentence_processor.py**: 単語の用例文自動生成

## 5. 技術的アピールポイント

### ポイント1: フルスタック開発経験
バックエンド（Django）、クラウドインフラ（GCP）、外部API統合（LINE・OpenAI）を一人称で設計・実装した経験を持ちます。

### ポイント2: セキュリティへの意識
LINE APIのシークレット情報をエンベロープ暗号化で保護し、Cloud KMSによる鍵管理まで実装しました。HSTS、CSRF保護、XSS対策も適切に設定しています。

### ポイント3: スケーラブルな設計
マルチテナントSaaSとして、組織・教室・学生の階層構造に対応したデータ設計とアクセス制御を実装しました。Google App Engineの水平スケーリングにも対応しています。

### ポイント4: AIプロダクト開発経験
ChatGPT（OpenAI API）を使った問題自動生成・会話応答システムを実用レベルで実装しています。プロンプトエンジニアリングから後処理まで一貫して担当しました。

### ポイント5: アルゴリズム実装力
SuperMemo-2スペースドリピティションアルゴリズムをDjangoモデルと連携する形で実装し、教育効果の高い学習体験を設計しました。

### ポイント6: 非同期・イベント駆動設計
Google Cloud Pub/SubとバックグラウンドリスナープロセスによるLINEプッシュ通知の非同期配信基盤を構築しました。
